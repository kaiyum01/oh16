Project: OH16 Small Business Solution
-------

This project mainly made by a company which made Digital Banner,poster etc.This project helps to business owner for many business relatated data records. That business has some process usch as Requisition,Order Confirmation,Challan Item and Bill processing.

So this project save all company, Item description, priceing calculation history. Create automatic generate Quotation ID with Quation page,Order confirmation with auto order code, Delivery Challan page creation with auto challan code and Bill processing or Invoice creation.

Every process you can print easily such as Quotation,Order,Challan and Bill 
 All data modified by Admin.
